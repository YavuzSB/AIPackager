🚀 AIPackagerAIPackager is a high-performance tool designed to optimize large software projects for AI assistant context limits (e.g., NotebookLM, Gemini, Claude). It intelligently scans your project, filters unnecessary files, and packages the content into manageable 400 KB chunks.✨ Key FeaturesSmart Filtering: Automatically excludes directories like .git, node_modules, and build, alongside binary/media files.400 KB Chunking: Splits project content into precise 400 KB parts while maintaining line and word integrity.Automatic Indexing: Generates a comprehensive INDEX.txt that showcases the included file tree and lists skipped items.Modern GUI: A sleek desktop interface powered by Dear ImGui and GLFW, featuring full Drag & Drop support.Powerful CLI: A dedicated command-line interface for advanced users and automation workflows.Native Cross-Platform: Built to run natively on Windows, Linux, and macOS with zero external runtime dependencies.📦 Download & InstallationNo installation required. Download the pre-compiled binary for your operating system from the Releases page.Windows: aipackager-gui-windows.exeLinux: aipackager-gui-ubuntu (Wayland & X11 compatible)macOS: aipackager-gui-macos🛠️ UsageDesktop (GUI)Launch the application.Drag and drop your project folder into the window.The tool automatically creates an ai_export folder within your target directory.Copy any chunk directly to your clipboard using the "Copy to Clipboard" buttons.Terminal (CLI)Bash# Basic usage example
./aipackager-cli --dir ./my_project --out ./export_folder --max-size 400000
⚙️ Technical ArchitectureThis project is built using Modern C++20 standards and follows strict SOLID principles.Core Logic: An isolated static library containing the scanner, packager, and chunk management logic.Memory Safety: Strict adherence to RAII and smart pointer usage ensures zero memory leaks.Performance: Utilizes std::filesystem for efficient tree traversal and streams files in chunks to minimize RAM usage.CI/CD: Automated multi-platform builds via GitHub Actions.📄 LicenseThis project is licensed under the MIT License.
